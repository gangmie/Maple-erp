# Maple ERP

> 메이플스토리 플레이를 위한 개인 관리 시스템

## 프로젝트 소개

Maple ERP는 메이플스토리를 플레이하면서 필요한 모든 정보를 하나의 시스템에서 관리하기 위한 프로젝트입니다.

단순한 메모장이나 체크리스트가 아니라, 데이터베이스를 중심으로 정보를 체계적으로 관리하고, 반복되는 작업은 자동화하는 것을 목표로 합니다.

이 프로젝트는 Notion을 데이터베이스로 사용하고, 메이플스토리 Open API와 GitHub Actions를 연동하여 캐릭터 정보를 자동으로 관리할 수 있도록 설계됩니다.

---

## 프로젝트 목표

- 메이플 플레이 정보를 하나의 시스템에서 관리
- 반복적인 작업 자동화
- 직접 입력해야 하는 정보 최소화
- 데이터 기반의 플레이 기록 및 통계 관리
- 장기간 사용 가능한 구조 설계

## 프로젝트 철학

이 프로젝트는 단순한 노션 템플릿을 만드는 것이 아닙니다.

'메이플스토리를 오래 플레이하는 사람을 위한 개인 ERP 시스템'을 목표로 합니다.

구조를 먼저 설계하고, 자동화를 적극 활용하여 입력은 최소화하면서도 필요한 모든 정보를 한곳에서 관리할 수 있도록 개발합니다.
